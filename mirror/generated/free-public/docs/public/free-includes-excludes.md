# Elementify Free Includes / Excludes

This page is the short public summary for the mirror-safe Free surface.

## Free includes

- scenario-first front door and intent routing
- runtime wizard families for new-site-lite, stack-bootstrap, optimization-lite, relaunch-lite, and extension-lite
- limited guided transitions for the first safe Free paths
- site assessment and context capture
- prioritized recommendations
- brand setup
- Creator Light and simple assembly
- local Elementor Library operations
- honest write validation
- public quickstart and release guidance

## Free excludes

- Advanced-only premium library flows
- Theme Builder workflows
- brand adaptation planning
- output critique and repair guidance
- Studio orchestration
- cloud library semantics
- cross-site delivery promises

## Public rule

Free must stay public, mirror-safe, and useful on its own. It should feel like a real operational lift for the current site, not like a teaser for private features.
It must also remain a credible 360-degree starting point for users who are not yet fully in the Elementor stack.
