# Elementify Free Quickstart

## Purpose

This is the canonical public quickstart for the mirror-safe Free surface.

Free is the public entry point for Elementify:

- public and mirror-safe
- local-site focused
- operationally useful without premium assets or cloud services

## Install

1. Install the WordPress plugin from the repo's `plugin/` directory.
2. Activate the plugin in WordPress admin.
3. Generate a site API key in the Elementify MCP settings.
4. Install the MCP server locally with `npm install -g @elementify/mcp`.
5. Configure your site in `~/.elementify/config.json`.
6. Connect the MCP server to your client.

## Use

Start with the Free workflows that are public and mirror-safe:

- scenario-first front door and intent routing
- runtime wizard families for:
  - new-site-lite
  - stack-bootstrap
  - optimization-lite
  - relaunch-lite
  - extension-lite
- limited guided transitions for the first safe bootstrap and optimization paths
- site assessment and context capture
- prioritized recommendations
- brand setup
- Creator Light
- local Elementor Library operations
- honest write validation

Free is meant to work both as:

- a 360-degree starting point for people who do not yet have Elementor in place
- a guided uplift layer for existing WordPress + Elementor sites

## What Free is not

Free does not include:

- Advanced-only premium library flows
- Studio orchestration
- cloud library semantics
- cross-site delivery promises
